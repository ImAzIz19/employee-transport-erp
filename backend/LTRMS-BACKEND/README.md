# LTRMS-BACKEND

This project is a **Spring Boot backend application** designed to support the LTMS system.

To fully run and configure the project, please refer to the official setup documentation available at the following link:

👉 [Setup Guide on Google Docs](https://docs.google.com/document/d/1Uy69fuzRqn1mYQ67Z9m-0UXF3F2fR5XYA-jH5JfEyPs/edit?usp=sharing)

Make sure to have **Java**, **Maven**, and **PostgreSQL** installed and configured correctly as described in the guide.
